# VEKTRA - Auto Parts Platform

VEKTRA is a premium automotive aftermarket platform designed for the Albanian market. It connects individual buyers and professional garages with authorized scrapyards (graveyards) and spare part sellers.

## Features
- **Advanced Search**: Filter parts by Brand, Model, and Part Type.
- **Direct Contact**: Instant connection via WhatsApp, Phone, or Email.
- **Admin Dashboard**: Comprehensive management of users, parts, and security settings.
- **Premium UI**: Modern dark-themed design with high-performance animations.

## Tech Stack
- **Framework**: Next.js 16 (App Router)
- **Database/Auth**: Supabase
- **Styling**: Tailwind CSS 4.0
- **Icons**: Lucide React
- **Security**: Cloudflare Turnstile CAPTCHA

## Getting Started

1. **Environment Variables**: Create a `.env.local` with your Supabase and Turnstile credentials.
2. **Install Dependencies**: `npm install`
3. **Run Dev Server**: `npm run dev`
4. **Build**: `npm run build`
